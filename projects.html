SOC Case Studies

Below are selected real-world security incidents and investigations that I have handled as part of SOC operations.

Brute Force Authentication Attack – Windows Active Directory

Context: Multiple failed login attempts followed by a successful authentication on a critical server.

My Role: SOC Analyst – Investigation & Response

Actions Taken:

  • Analyzed Windows Event IDs 4625 and 4624 to identify attack pattern
  • Correlated source IP behavior and login timestamps
  • Checked endpoint activity using EDR telemetry
  • Validated IP reputation using OSINT sources
  • Recommended account reset, MFA enforcement, and IOC blocking

Outcome: Account secured, attack contained, detection rule improved.

Suspicious PowerShell Execution – Encoded Command

Context: Alert triggered for encoded PowerShell execution originating from an Outlook process chain.

My Role: SOC Analyst – Threat Analysis

Actions Taken:

  • Reviewed EDR process tree and command-line arguments
  • Identified Base64-encoded PowerShell execution
  • Checked persistence mechanisms and scheduled tasks
  • Isolated endpoint and collected forensic artifacts

Outcome: Endpoint secured, no lateral movement observed, controls hardened.

Malware Detection & Endpoint Containment

Context: Endpoint security alert indicating suspicious executable behavior.

My Role: SOC Analyst – Incident Handling

Actions Taken:

  • Analyzed malware behavior via EDR telemetry
  • Checked file hash and execution context
  • Performed endpoint isolation and quarantine
  • Verified removal of persistence artifacts

Outcome: Malware eradicated, system restored to secure state.
